# Changelog

## [1.1.0] - 2026-01-26

### Features
- Statistics dashboard — track focus time, breaks taken, and streaks
- Global keyboard shortcuts (⌘⇧N new session, ⌘⇧P pause/resume, ⌘⇧B skip break, ⌘⇧S open stats)
- Health exercises and stretches during breaks
- Data pruning for long-term storage

### Improvements
- Simplified stats and break screen UI
- Integrated exercises directly into the break screen

## [1.0.0] - 2026-01-24

### Features
- Fullscreen break reminders following the 20-20-20 rule
- Configurable session length (15-50 min) and break duration (20s-5 min)
- Long breaks after multiple sessions
- Menu bar timer with session countdown
- Skip and snooze controls
- Launch at login
- Homebrew cask installation
- Landing page at afk-app.vercel.app
